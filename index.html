    My mission

    The mission of More Than Code, my mission, is to improve the quality of software. When the quality of the codebase improves, the quality of the software improves as well. This results in users being more enthousiastic to work with the product. I strive to accomplish this mission through a number of disciplines which define not just a professional developer, but define a software craftsman (or craftswoman). By adhering to these principles, it is my goal to build productive partnerships with both colleagues and clients.

    Clean code

    I deliver properly structured, well documented and thoroughly tested code that is easy to change and extend.

    Continuous improvement

    Whether it's inefficient, hard to read code or a gap in my own knowledge, I strive to improve myself, the codebase and the infrastructure.

    Find solutions

    There's is no problem that cannot be solved. I will look both inside and outside the box to find an elegant solution.

    Engineering discipline

    It takes effort to structure a complex program. Every part needs to be cared for equally. I impose that structure into a lot of parts of my life, but nowhere more than in my code.

    Community contribution

    Throughout my carreer, I have learned a lot from what others put freely out there. I hope that what I give back will benefit othes as I have received in the past.
