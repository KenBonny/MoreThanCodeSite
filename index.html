My mission

The mission of More Than Code, my mission, is to improve the quality of software. When the quality of the codebase improves, the quality of the software improves as well. This will result in users being more enthousiastic to work with the product. I will strive to accomplish this mission through a number of disciplines which define not just a professional developer, but also a software craftsman (or craftswoman)

Clean code

Deliver properly structured, well documented and thoroughly tested code

Flexible code

Create code that is easy to change and extend

Improve code

Improve inefficient code

Keep learning

Continuously improve myself and learn about new ways

Give back

Contribute to a community of professionals

By adhering to these principles, it is my goal to build productive partnerships with both colleagues and clients.
