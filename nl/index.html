Andere informatie

Op mijn blog schrijf ik over de technische uitdagingen die ik tegenkom en hoe ik deze oplos. Ik hoop dat het mede-ontwikkelaars helpt. Ik schrijf occasioneel ook mijn persoonlijk leven of over situaties die ik meemaak.

Mijn blog is een goede plek om mijn technische vaardigheden in te schatten, maar er gaat niets boven de effectieve code die ik schrijf. Deze kan op mijn GitHub profiel gevonden worden. Hier kan u zien of mijn code effectief zo goed is als ik beweer.

Geen profiel is compleet zonder een CV .

Als laatste heb ik ook een KeyBase profiel . Dit kan u gebruiken om een beveiligd bericht naar mij te versturen. Als u niet weet wat dit is, dan kan u dit gewoon negeren.
